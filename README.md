# linux-fundamentos-seguranca 
Este repositório tem o objetivo de consolidar meus conhecimentos  de Linux com foco em segurança,abordandando conceitos essenciais  para a certificação LPIC-1 e para atuação prática na área. 

## Contéudos
- Usuários e grupos
- Permissões de arquivos
- Processos e serviços
- Logs do sistema
- Firewall básico.

## Ambiente
- Linux ( Debian / Ubuntu)
- Virtual Box
